# 技术雷达 - 暂缓 (Hold)

**最后更新**: 2025-11-30
**评审周期**: 每季度

---

## 概述

"暂缓"象限包含不推荐在新项目中使用的技术。现有项目可以继续使用，但应规划迁移。

---

## 暂缓使用的技术

| 技术 | 替代方案 | 暂缓原因 |
|------|----------|----------|
| **Java 8/11** | Java 17/21 | 安全更新即将终止 |
| **Maven** | Gradle | 构建性能和灵活性 |
| **JavaScript (无类型)** | TypeScript | 类型安全要求 |
| **jQuery** | React/Vue | 现代化前端架构 |
| **MongoDB (通用场景)** | PostgreSQL | 数据一致性要求 |
| **Jenkins** | GitHub Actions | 维护成本和现代化 |
| **Moment.js** | date-fns / Day.js | 包体积和维护状态 |

## 迁移指南

### Java 8/11 → Java 17/21

```markdown
1. 检查依赖兼容性
2. 更新构建配置
3. 处理废弃API
4. 测试验证
5. 渐进式部署
```

### Maven → Gradle

```markdown
1. 使用 gradle init 生成初始配置
2. 迁移依赖声明
3. 配置插件
4. 验证构建结果
5. 更新 CI/CD 配置
```
