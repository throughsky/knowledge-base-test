# 术语词典

## 稳定币相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 稳定币 | Stablecoin | 与法币或资产锚定的数字代币 |
| 铸造 | Mint | 创建新的稳定币 |
| 赎回 | Redemption | 销毁稳定币换回法币 |
| 储备金 | Reserve | 支撑稳定币价值的底层资产 |
| 储备率 | Reserve Ratio | 储备金/发行量，需>=100% |
| 锚定 | Peg | 稳定币与法币的价格挂钩 |
| 脱锚 | Depeg | 价格偏离锚定值 |

## RWA相关

| 术语 | 英文 | 定义 |
|------|------|------|
| RWA | Real World Assets | 现实世界资产 |
| 代币化 | Tokenization | 将资产权益转化为链上代币 |
| STO | Security Token Offering | 证券型代币发行 |
| 合格投资者 | Accredited Investor | 符合监管要求的投资者 |
| 白名单 | Whitelist | 允许持有/交易代币的地址列表 |
| 分红 | Dividend | 资产收益分配 |
| 估值 | Valuation | 资产价值评估 |
| SPV | Special Purpose Vehicle | 特殊目的载体 |

## 通用Web3

| 术语 | 英文 | 定义 |
|------|------|------|
| 智能合约 | Smart Contract | 链上自动执行的代码 |
| Gas | Gas | 链上交易手续费 |
| 交易哈希 | Transaction Hash | 链上交易唯一标识 |
| 钱包地址 | Wallet Address | 用户的链上账户地址 |
| 多签 | Multi-sig | 需多方签名的钱包/交易 |
| AML | Anti-Money Laundering | 反洗钱 |
| KYC | Know Your Customer | 客户身份验证 |

## 金融通用

| 术语 | 英文 | 定义 |
|------|------|------|
| 抵押品 | Collateral | 用于担保的资产 |
| 清算 | Liquidation | 强制出售抵押品 |
| 托管 | Custody | 资产保管服务 |
| 合规 | Compliance | 符合监管要求 |
