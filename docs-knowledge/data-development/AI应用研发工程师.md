#### **职位名称：AI应用研发工程师 3-4人**

**【实习项目概述】** 使用AI和构建AI工具，探索在研发领域，在产品，开发，测试，持续集成等研发全生命周期环节中赋能研发团队的项目。

不限于以下方面的职责：

* **前沿技术调研与原型开发**：追踪 LLM（大语言模型）、AI Coding（如 Claude Code, Cursor 机制）、Autonomous Agents（自主智能体）,LangGraph, RAG等领域的最新进展，快速构建 Proof of Concept (PoC) 原型。
* **AI Coding 工具研发**：参与CodeWiki、自动化Code Review工具的使用和开发，测试自动化，Agent应用开发。
* **垂直领域知识库构建**：负责私有化知识库的体系建设，熟悉当前多种框架和应用的多种检索召回策略优缺点。
* **工程化落地**：动手能力强，将调研成果转化为高质量的 Python/Java 代码。
* **Prompt Engineering 与 调优**：设计并优化复杂的 Prompt 流程（Chain/Graph），提升模型在特定编程场景下的代码生成质量和逻辑推理能力。

**【能力要求】**

* **本科及以上学历，计算机、数学或相关专业**。
* **具有极强的自驱力和好奇心，习惯通过阅读英文文档/论文/技术博客解决问题。**
* **Github活跃者优先，经常关注社区生态和技术网站。**
* **编程能力**：精通Python，熟悉常用 AI 开发框架（如 LangChain, LlamaIndex, Spring AI, Semantic Kernel）。熟悉 Java 或 TypeScript，具备良好的面向对象设计思想，有开发过大型项目，理解应用架构者优先。具备扎实的数据结构与算法基础。
* **AI 与数据工程**：熟悉主流 LLM API（OpenAI, Anthropic, Gemini）及国产大模型接口开发。
* **知识库**：理解 RAG（检索增强生成）架构，相关图算法，了解 GraphRAG 或基于知识图谱的增强生成者优先。
* **加分项**：深度使用者：重度使用 Cursor, GitHub Copilot,Claude Code 等 AI 工具，并有自己的一套实践方法论。
* **Agent 开发经验**：有过 AutoGPT, MetaGPT, LangGraph 等 Agent 开发或二开经验。

**【我们提供】探索 AI 2.0 时代最前沿技术（AI Coding + Agent）的机会。极客氛围，鼓励尝试新技术，允许在探索中试错。各种顶尖大模型Token管够，有机会参与企业级核心AI应用开发的机会。**




#### **职位名称：AI 应用研发工程师（实习）—— AI Native 工具方向**

**【项目/团队背景】**
我们需要你加入一个高度敏捷的极客团队，致力于探索 **AI Engineering** **的最前沿。我们的目标是构建下一代 AI Native 工具，将 AI 能力注入到研发全生命周期（产品、开发、测试、CI/CD）中，真正赋能研发团队，解放生产力。**

**【你将负责】**

* **前沿技术破壁**：紧跟 LLM（大语言模型）与 Agent 领域的最新进展（如 **Claude Code**, **Cursor**, **MCP**, **LangGraph** **等），快速验证想法并构建 Proof of Concept (PoC) 原型。**
* **AI Coding 工具链研发**：参与企业级代码知识库（CodeWiki）、自动化 Code Review、智能测试生成等工具的开发；探索从 Copilot 到 **Autonomous Agents**（自主智能体）的工程演进。
* **垂直领域 知识库体系构建**：负责私有化知识库的建设，深入研究检索召回策略，解决复杂上下文下的检索准确率问题。
* **Prompt Engineering 与 编排**：不仅仅是写 Prompt，而是设计复杂的 **Chain/Graph** **逻辑，通过工程化手段提升模型在特定编程场景下的推理与生成质量。**
* **全栈工程落地**：你需要具备极强的动手能力，能将调研成果转化为高质量的 **Python** **或** **Java** **生产级代码。**

**【我们需要这样的你】**

* **学历背景**：本科及以上学历，计算机、数学或相关理工科专业。
* **极客精神**：具有极强的自驱力，**习惯阅读英文**的一手技术文档、Paper 和技术博客，不满足于调包，喜欢探究原理。
* **社区活跃度**：GitHub 活跃用户优先，或是 Hacker News、Twitter/X 技术圈的常客。
* **硬核编程能力**：

  * **Python (Must)**：熟悉Python，熟悉 LangChain, LlamaIndex, Semantic Kernel 等框架源码者优先。
  * **Java/TypeScript (Plus)**：熟悉 Java (Spring AI/Spring Boot) 或 TS，具备良好的 OOP 设计思想，能理解复杂的企业级应用架构。
  * **具备扎实的数据结构与算法功底。**
* ****超级加分项**：**

  * **熟悉主流 LLM API（OpenAI, Anthropic, Gemini）及国产模型接口。**
  * ****Power User**：重度使用 Cursor, Claude Code, GitHub Copilot 等工具，并有一套自己的最佳实践方法论。**
  * ****Agent 实战**：有过 LangGraph, AutoGPT, CrewAI 等 Multi-Agent 框架的开发或二开经验。**
  * **熟悉知识库或使用过RAG** ：理解知识库构建方法论或实践过RAG ，熟悉向量数据库（Milvus/Chroma/ES），了解 GraphRAG或图算法者极大加分。

**【我们提供】**

* **探索未来**：接触 AI 2.0 时代最前沿技术（AI Coding + Agent ）的机会。
* **极客氛围**：鼓励尝试新技术，宽容失败，扁平化管理，Talk is cheap, show me the code。
* **资源支持**：**顶尖大模型 Token 管够**，高性能计算资源支持，有机会参与企业级核心 AI 应用的从 0 到 1。
